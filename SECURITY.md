# Security Policy

## Supported Versions

Please use the latest version of Quorlyx. Only the most recent release is officially supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| > 1.0.0 | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Quorlyx, please do not disclose it publicly.

Instead, please report it via GitHub's private vulnerability reporting feature on our repository or contact us directly. We will validate and address the issue as quickly as possible.
