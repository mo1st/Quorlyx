# Knowledge Base Upload

Quorlyx can use uploaded Knowledge Base files as context for AI responses.

## Supported Workflow

1. Prepare concise business, product, policy, or support documents.
2. Upload supported files through the WordPress admin interface.
3. Select which files should be used for AI context.
4. Test responses with realistic visitor questions.

## Recommended Content

- Product or service descriptions
- Store policies
- FAQ answers
- Shipping and return information
- Booking or lead-capture instructions
- Internal setup notes that are safe for AI context

## Safety

- Do not upload files containing passwords, payment data, private customer records, or production credentials.
- Keep API keys outside Knowledge Base files.
- Tell the assistant to say when information is missing instead of inventing policies or guarantees.
- Review extracted text before using it in production.
