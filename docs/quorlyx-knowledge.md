# Quorlyx Knowledge Notes

Quorlyx is an open-source WordPress and WooCommerce plugin for behavior-based AI automation.

## Setup

1. Install and activate the plugin.
2. Open **Quorlyx > Settings**.
3. Add your own AI provider API key.
4. Choose a provider and model.
5. Configure the first trigger or chat workflow.

## Core Areas

- Chatbot settings
- Behavior triggers
- Knowledge Base files
- AI content engine
- Logs and diagnostics
- WooCommerce and conversion tracking

## Security

- Do not commit real provider API keys.
- Do not collect passwords, payment card numbers, or private account credentials in chat.
- Keep production credentials in server configuration or environment variables.
- Review uploaded Knowledge Base files before using them in AI context.

## Support Model

Community support happens through GitHub issues and Discord. Optional paid setup help can be offered separately from the open-source plugin.
