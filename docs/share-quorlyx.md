# Share Quorlyx

The best way to ask for GitHub stars is to show value first, then make a simple ask. Avoid begging, mass posting, or posting the same text everywhere. Good promotion sounds like: "Here is what I built, here is who it helps, here is the repo if it is useful."

## Short Star Ask

Use this when replying to someone who already found the project useful:

```text
Thanks for checking it out. If Quorlyx is useful for your WordPress or WooCommerce workflow, a GitHub star would help more developers find it:

https://github.com/mo1st/Quorlyx
```

## Reddit Post

```text
I open-sourced Quorlyx, an AI chatbot and behavior automation plugin for WordPress and WooCommerce.

Most chatbot tools wait for the visitor to ask something. Quorlyx can react to site behavior instead: product views, scroll depth, exit intent, cart activity, checkout drop-off, return visits, page context, conversion goals, and WooCommerce product data.

It runs inside WordPress and uses your own AI provider key, so there is no required hosted chatbot subscription. It supports providers like OpenAI, Gemini, Claude, Mistral, Grok, and DeepSeek.

Useful for:
- WooCommerce cart recovery
- Product-aware AI chat
- Behavior-based triggers
- Knowledge Base answers from posts, pages, products, and uploaded files
- Conversion goals and A/B testing
- Content Insights and SEO/content tools

GitHub: https://github.com/mo1st/Quorlyx
Website: https://quorlyx.dev

I would appreciate feedback from WordPress developers, store owners, and agencies. If the project looks useful, a GitHub star also helps other developers discover it.
```

## Good Places To Share

- WordPress developer communities.
- WooCommerce store-owner communities.
- Indie hacker and open-source project communities.
- Agency and freelancer communities.
- AI automation communities.
- Reddit threads where people ask for WordPress chatbot, WooCommerce automation, cart recovery, or AI Knowledge Base tools.

## How Not To Ask

- Do not DM strangers only asking for stars.
- Do not post the same message repeatedly.
- Do not overclaim that the project replaces every chatbot.
- Do not ask before showing what the plugin does.
- Do not hide that it is open source and BYOK.

## Best Ask Formula

```text
I built [what it is] for [who it helps].
It solves [specific problem].
Here is what it can do: [3-5 concrete features].
Repo: [GitHub link]
If it is useful, a star helps more people discover it.
Feedback is welcome.
```
