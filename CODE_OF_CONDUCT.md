# Code Of Conduct

Quorlyx is an open-source project for WordPress, WooCommerce, and AI-assisted site automation. The community should stay practical, respectful, and useful for developers, store owners, agencies, and contributors.

## Expected Behavior

- Be respectful and patient when discussing bugs, ideas, setup problems, and code reviews.
- Share clear technical details, reproduction steps, screenshots, logs, or examples when they help.
- Assume good intent, but keep feedback direct and focused on the project.
- Respect privacy. Do not post API keys, private customer conversations, order exports, database dumps, or private site credentials.
- Keep discussions relevant to Quorlyx, WordPress, WooCommerce, AI providers, automation workflows, documentation, or related integrations.

## Unacceptable Behavior

- Harassment, threats, hate speech, insults, or personal attacks.
- Publishing another person's private information or credentials.
- Posting AI provider keys, webhook secrets, customer records, private logs, or private exports.
- Spam, unrelated self-promotion, or repeated off-topic posting.
- Deliberately misleading bug reports, pull requests, or support requests.

## Reporting Problems

If you need to report conduct problems, contact the project owner through the website:

`https://quorlyx.dev`

For serious security issues, follow [SECURITY.md](SECURITY.md) instead of posting details publicly.

## Enforcement

Maintainers may edit or remove comments, close issues or pull requests, limit participation, or block users when behavior makes the project less safe or less useful for the community.
