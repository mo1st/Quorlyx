# Quorlyx Open Source Project

Local staging folder for preparing Quorlyx as an open-source WordPress/WooCommerce automation plugin.

## Model

- Plugin core is free and open source.
- GitHub issues and Discord handle community support.
- Optional setup support is a one-time $20 pass for 14 days.
- Future paid work can include custom installation, trigger customization, WooCommerce/CRM/webhook integrations, and a cloud version.

## Before Publishing

- Remove API keys, secrets, test data, private links, emails, and commercial-only logic.
- Prepare a strong README with screenshots, installation, architecture, AI workflow, webhooks, security notes, and roadmap.
- Present Quorlyx as an open-source AI behavior automation plugin for WordPress and WooCommerce.
